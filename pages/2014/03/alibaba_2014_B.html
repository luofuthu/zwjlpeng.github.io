A: h(K)=K/N;

B: h(K)=1;

C: h(K)=K mod N;

D: h(K)=(K+rand(N)) mod N, rand(N)返回0到N-1的整数

A: 堆排序

B：插入排序

C: 冒泡排序

D:快速排序

3 下面说法错误的是

A: CISC计算机比RISC计算机指令多

B: 在指令格式中，采用扩展操作码设计方案的目的是为了保持指令字长不变而增加寻址空间

C:增加流水线段数理论上可以提高CPU频率

D:冯诺依曼体系结构的主要特征是存储程序的工作方式

A:CPU

B: Cache

C:RAM

D:ROM

A:DECBA

B:DCEBA

A：汇编

A：static成员变量在对象构造时候生成

B: static成员函数在对象成员函数中无法调用

C: 虚成员函数不可能是static成员函数

9：某进程在运行过程中需要等待从磁盘上读入数据，此时进程的状态将：

A: 从就绪变为运行

B:从运行变为就绪

C: 从运行变为阻塞

D:从阻塞变为就绪

Int f(unsigned int n)
{

If(n==0||n==1)

Return 1;

Else

Return n*f(n-1);

}

A: O(1)

B:O(n)

C:O(N*N)

D:O(n!)

A:18

B:24

C:21

D；不可能

C: n-1

D:4

A: O(n),O(n)

B:O(n),O(1)

C:O(1),O(n)

D:O(n),O(n)

Struct A

A: 颜色全相同

B:颜色全不相同

C:颜色全相同

D:颜色无红色

A: 每张牌出现在n个位置上的概率相等

B: 每张牌出现在n个位置上的概率独立

C: 任何连续位置上的两张牌的内容独立

A: 10

X和y是全局变量，初始为0。以下哪一个是r1和r2的可能值：

A: r1=1,r2=1

B: r1=1,r2=0

C:r1=0,r2=0

D:r1=0,r2=1

A: 通过就绪和运行的进程数来反映

C: 通过uptime查看
