7、What is the time complexity of the following function?()

int foo(int N)
{
    int result=0;
    for(int i=1;i<=N;++i)
    {
        for (int j=1;j<=N;j+=i)
        {
            result+=i+j;
        }
    }
    return result;
}

A O(N) B O(NlogN) C O(N^1.5) D O(N^2)
